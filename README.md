# Мини-Склад — учебная конфигурация 1С

Учебный проект в рамках стажировки КБ99.

## Что внутри

**Справочники:**
- Товары
- Контрагенты

**Документы:**
- Поступление — приход товара на склад
- Реализация — продажа товара клиенту

**Подсистема:** Мини-Склад

## Как открыть

1. Установить 1С:Предприятие 8.3
2. Добавить новую базу без конфигурации
3. В Конфигураторе: Конфигурация → Загрузить конфигурацию из файлов → выбрать папку `praktika`
4. Обновить конфигурацию базы данных (F7)
5. Запустить в режиме 1С:Предприятие
